# Language posture

{{LANGUAGE_POSTURE_DECISION}}

This governs how every spec file expresses logic. `spec-1-specify`,
`spec-2-grill`, `spec-3-architect`, and `spec-4-edit` all honour it when they
write or revise spec prose, so the whole spec reads with one voice instead of
drifting file by file. If the project's needs change, revisit the posture
*here* (supersede this ADR) rather than quietly changing it in one file.

# Natural Writing Craft

For when a draft is tell-free but still reads flat. The tells catalog says
what to remove; this says what makes prose alive. All of it defers to the
user's profile (`~/.claude/rules/write-like-me.md`) — these defaults exist
only for the space the profile doesn't cover.

## Rhythm

- Paragraphs are beats, not containers: a one-sentence paragraph lands a
  point; a long one builds a case. Same-size paragraphs in a row means the
  pacing was never considered.
- Read the draft at speaking pace. Anywhere you'd never breathe, or breathe
  twice the same way in a row, rewrite.

## Concreteness

- One concrete anchor per paragraph is the floor: a number, name, date,
  path, or quote. Not "communication challenges" but "the thread went quiet
  for two days."
- If a sentence would be true of any project or team, it says nothing about
  this one. Cut it or sharpen it.

## Commitment

- Precise uncertainty reads as expertise ("holds above 1M docs, untested
  below"); vague uncertainty ("may potentially") reads as evasion. Put the
  hedge in the claim's content, not its framing.

## Texture

- Small asymmetries are human: a parenthetical aside, a deliberate fragment,
  a colloquialism inside formal prose. One per section, not one per line.
- Transition by content, not connective: the next paragraph's first noun
  should link to the last one's idea. When that linkage exists, "Moreover"
  has nothing to do.
- If the piece exists to make one point, write the ending first; everything
  that doesn't feed it is a candidate cut.

# Natural Writing Craft

Techniques for when a draft is tell-free but still reads flat. The tells
catalog says what to remove; this says what makes prose alive.

## Rhythm

- **Burstiness is the master variable.** Statistical detectors measure exactly
  this: humans vary sentence length and structure; models regress to the mean.
  Write a long, winding sentence that earns its length. Then stop short.
- Paragraphs are beats, not containers. A one-sentence paragraph lands a
  point. A long paragraph builds a case. Same-size paragraphs in a row means
  the pacing was never considered.
- Read the draft in your head at speaking pace. Anywhere you'd never breathe,
  or breathe twice the same way in a row, rewrite.

## Concreteness

- Swap category words for instances: not "communication challenges" but "the
  thread went quiet for two days." Specifics are the single strongest
  humanity signal because a model defaults to the safest abstraction.
- Numbers, names, dates, file paths, quotes. One concrete anchor per paragraph
  is a good floor.
- If a sentence would be true of any project/product/team, it's saying
  nothing about this one. Cut it or sharpen it.

## Commitment

- Have one opinion per piece minimum. "X is the wrong default here" is human;
  "X has both advantages and disadvantages" is filler.
- State the claim, then the evidence. Hedging belongs in the claim's *content*
  ("this holds for corpora over 1M docs, untested below") — precise
  uncertainty reads as expertise; vague uncertainty ("may potentially") reads
  as evasion.

## Texture

- Small asymmetries are human: a parenthetical aside, a sentence fragment used
  on purpose, a colloquialism inside otherwise formal prose. One per section,
  not one per line.
- Transitions by content, not connective: the next paragraph's first noun
  should link to the last one's idea. When that linkage exists, "Moreover" has
  nothing to do.
- Write the ending first if the piece exists to make one point. Everything
  that doesn't feed that ending is a candidate cut.

## The voice override

All of the above defers to the user's profile (`~/.claude/rules/write-like-me.md`).
If the profile says they write in clipped fragments, or they love semicolons,
or they never open with a greeting — the profile wins. These defaults exist
for the space the profile doesn't cover.
